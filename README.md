# Проект "Портфолио Хаб" ?!

## Реализация 

Поскольку современные реалии постоянно меняются, а сфера программирования развивается ускоренными темпами, приходится осваивать новые технологии, такие как искусственный интеллект.

Этот проект был реализован с использованием инструментов искусственного интеллекта, без ручной разработки кода с моей стороны. Я ставил задачи, проектировал структуры решения и управлял всеми процессами разработки, включая контроль качества и финальное принятие решений

Я опробовал формат vibe‑кодинга — и вот что получилось 

## Quick Start

```bash [Terminal]
npm create nuxt@latest -- -t github:nuxt-ui-templates/starter
```

## Setup

Make sure to install the dependencies:

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```
